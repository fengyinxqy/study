## 📝 学习方式

本笔记采用 **苏格拉底式提问法**：

1. 🤔 **先思考** —— 阅读问题后暂停，尝试用自己的理解回答
2. 💡 **再看答案** —— 对比自己的理解和正确解释
3. ✍️ **动手实践** —— 在实际环境中执行命令或代码验证
4. 📖 **记录心得** —— 把新的理解记录到笔记中
