## 📚 推荐学习资源

### 书籍
- 《MySQL 必知必会》- 入门首选
- 《高性能 MySQL》- 进阶必读
- 《数据库设计那些事》- 设计实践

### 在线资源
- [MySQL 官方文档](https://dev.mysql.com/doc/)
- [SQLZOO](https://sqlzoo.net/) - SQL 在线练习
- [LeetCode Database](https://leetcode.com/problemset/database/) - 数据库算法题

### 工具
- [MySQL Workbench](https://www.mysql.com/products/workbench/) - 官方图形化工具
- [DBeaver](https://dbeaver.io/) - 免费通用数据库工具
- [dbdiagram.io](https://dbdiagram.io/) - 在线 ER 图绘制
